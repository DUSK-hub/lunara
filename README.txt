# Lunara

Lunara is a Flask-based web application built to help students in my college.  
Currently, it allows users to upload PDF files and automatically extract text from them.

---

## Features (Current)

- Upload PDF files
- Extract text from uploaded PDFs

---

## Planned Features (Upcoming)

- AI-powered text summarization
- Automatic quiz generation from summaries
- Dashboard for subjects and study materials
- Multi-user authentication

---

## Tech Stack

- **Backend:** Flask (Python)
- **AI Integration:** Currently using DeepSeek API (temporary — may change later)
- **Database:** SQLite
- **Frontend:** HTML templates (Jinja2), CSS, JS
