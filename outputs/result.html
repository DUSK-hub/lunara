### 1. Concise Summary

Discrete Mathematics is the study of mathematical structures that are fundamentally countable or distinct, unlike continuous systems such as calculus. It provides the essential theoretical foundation for computer science, focusing on logic, algorithms, and data structures. Key areas include formal logic for reasoning, set theory for grouping objects, and combinatorics for counting. Mastery of these topics is crucial for designing efficient software, securing digital communications, and optimizing complex systems.

**1. In logic, what is the converse of the conditional statement "If it is raining, then the ground is wet"?**
   a) If it is not raining, then the ground is not wet.

   c) The ground is wet if and only if it is raining.

**3. Which of the following describes a function that assigns each element in the domain to a unique element in the codomain?**
